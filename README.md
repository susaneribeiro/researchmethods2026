The first dataset was also opened in Rstudio to perform (1) data cleaning and (2) calculation of the scores for each construct.
During data cleaning, participants that did not participate in the posttest nor provided consent were excluded. 
The final sample consisted of 18 participants: 6, 8, 9, 15, 16, 18, 19, 21, 22, 24, 25, 28, 30, 33, 36, 37, 38, 39. 
The R script wrote the dataset: "dataset_clean.csv". This dataset was used in JASP to run the analyses to answer the research questions. 
Additionally, R script was also used for visualization with the creation of the simple plot and the Johnson Neyman chart.
